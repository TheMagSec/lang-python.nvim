# lang-python.nvim

A Python language bundle for Neovim.

----

## Installation

Use any Neovim compatible package manager, like:

```lua
vim.pack.add("https://github.com/themagsec/lang-python.nvim")
```

> [!WARNING]
> This package is still in development and may not work as expected.
> Custom configuration via `setup` function is not yet fully implemented.
> The default LSP's currently used are: `ruff` and `ty`
